PROJECT WOMBAT

What is Project Wombat?

Project Wombat was designed with you   in mind.

With how difficult the job market has become in recent years, a reliable data source would be ideal for use in hunting.
However, many sites nowadays are outdated, depreciated, and wholly confusing to filter through.
That's where Project Wombat comes in.
Our site operates by using the Bureau of Labor Statistics API, pulling their data and storing it in an easy and convenient location.
The end goal was to create an interactive and responsive Google Map, capable of displaying data based on location whether it be county or state.
From there, our Google Map would update accordingly to each location's data set, changing color to represent high and low data values.
